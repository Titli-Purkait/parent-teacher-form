Parent Teacher Conference Form

A responsive and visually appealing HTML/CSS form designed for parents or guardians to submit student information and preferred contact methods for scheduling parent-teacher conferences.

Features:

Clean, modern design with a semi-transparent container and box-shadow effect.

Custom-styled radio buttons with smooth animations.

Responsive layout suitable for desktop and mobile devices.

Clear sections for student information, parent/guardian information, preferred contact method, and additional notes.

Interactive submit button with hover effects.

Technologies Used:

HTML5

CSS3 (including pseudo-elements and custom form styling)
